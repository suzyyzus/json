常见问题
这个工具会上传我的 JSON 吗？

不会。格式化与校验逻辑都在浏览器本地执行，不会主动发起网络请求。

为什么提示 JSON 不合法？

请检查引号、逗号、括号是否匹配。工具会尽量给出错误附近的行列位置，方便定位。

如何接入广告代码？

把 AdSense 的 client 与广告位 slot 填入页面配置后即可渲染（已预留顶部、侧边栏、底部广告位）。
